Przed uruchomieniem aplikacji należy w pliku src/main/resources/application.properties
wpisać następujące parametry dotyczące serwera lokalnego MySQL:

spring.datasource.username=<tutaj należy wpisać nazwę użytkownika>
spring.datasource.password=<tutaj należy wpisać hasło użytkownika>

Następnie należy uruchomić aplikację za pomocą przycisku "Run" z poziomu IntelliJ IDEA.
